Humble Light-Box

A simple lightbox plugin made with JQuery

View on GitHub

Demo

Instructions

Download the plugin, extract it and move it to your project folder


Link the relative css and js files in your html document

    <link rel="stylesheet" href="path/humbleLightBox.css">
    <script src="path/humbleLightBox.js"></script>

You also need JQuery and a custom version of JQuery-mobile, which are included in the download file

    <script src="path/jquery.mobile.custom.min.js"></script>

To obtain the light-box effect you just need to add the class "humbleLightbox" on the desired image.
